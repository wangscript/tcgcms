添加一个mc作为皮肤
function MovieSkin(url:String, bDontAnimate:Boolean, bSmooth:Boolean)
url: URL to load
bDontAnimate: Boolean; if true (the default in Sandy 1.2) we DISABLE the automatic update of the texture property.
bSmooth: Boolean; if true perform smoothing (performance-intensive).


填充面
function SimpleColorSkin(c:Number, a:Number)
c:The color ot the line
a:The alpha of the line


添加图片作为皮肤
function TextureSkin(t:BitmapData, bSmooth:Boolean, alpha:Number)
t: The actionScriptLink of the bitmap;
bSmooth: Boolean; if true perform smoothing (performance-intensive).
alpha:alpha

混合型皮肤
function MixedSkin(cb:Number, ab:Number, cl:Number, al:Number, tl:Number)
cb:Number The color used to fill the Object faces
cl:Number The color of the lines used to draw the Object edges
al:Number The alpha of the lines used to draw the Object edges
tl:Number The thickness of the lines used to draw the Object edges

移动摄像机
===============================================

moveSideways
function moveSideways(d:Number):Void
Allow the camera to translate along its side vector. If you imagine yourself in a game, it would be a step on your right or on your left
d:Number Move the camera along its side vector

moveUpwards
function moveUpwards(d:Number):Void
Allow the camera to translate along its up vector. If you imagine yourself in a game, it would be a jump on the direction of your body (so not always the vertical!)
d:Number Move the camera along its up vector

moveForward
function moveForward(d:Number):Void
Allow the camera to translate along its view vector. If you imagine yourself in a game, it would be a step in the direction you look at. If you look the sky you will translate upwards ! So be careful with its use.
d:Number Move the camera along its viw vector

moveHorizontally
function moveHorizontally(d:Number):Void
Allow the camera to translate horizontally If you imagine yourself in a game, it would be a step in the direction you look at but without changing your altitude.
d:Number Move the camera horizontally

moveVertically
function moveVertically(d:Number):Void
Allow the camera to translate vertically If you imagine yourself in a game, it would be a jump strictly vertical.
d:Number Move the camera vertically

移动立方体
===========================================

translate
function translate(tx:Number, ty:Number, tz:Number):Void
Realize a translation on all the 3 directions
tx:Number the amount of translation in x axis
ty:Number the amount of translation in y axis
tz:Number the amount of translation in z axis

var bg:Group = new Group();
var o:Object3D = new Box( 200, 200, 400 ,'tri');
var tg:TransformGroup = new TransformGroup();
var translation:Transform3D = new Transform3D();
translation.translate(x, y, z);
tg.setTransform(translation);
tg.addChild(o);
bg.addChild(tg);
